### `Import the library` ###
To get the Library into your project, Add jitPack to your build.gradle :
```gradle
allprojects {
	repositories {
		..
		maven { url 'https://jitpack.io' }
	}
}
  
 ```
Then add the dependency : 
```gradle
dependencies {
  implementation 'com.github.TomLeCollegue:ProgressBar-Library-Android-Kotlin:0.1.1'
}
  
 ```
